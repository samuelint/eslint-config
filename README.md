# Samuel Magny - eslint-config

My goto eslint config

## How to use

### 1. Install

##### pnpm

```
pnpm add -D @samuelint/eslint-config
```

### 2. Configure

```
# .eslintrc.json
{
    "extends": "@samuelint/eslint-config",
    ...
}
```

OR

```
# .eslintrc.js
{
    extends: "@samuelint/eslint-config",
    ...
}
```

### 3. Enjoy
